# 📄 Lite Article Exporter - 安装指南

## 🎯 产品简介
这是一个 Chrome 浏览器扩展，支持：
- ✅ 一键提取微信公众号或网页正文
- ✅ 多格式导出：Markdown、PDF、Word、纯文本
- ✅ 下载记录查询与快速重用
- ✅ 本地化存储，数据不出浏览器

---

## 🚀 安装方法（开发者模式）

### 📋 前提条件
- Google Chrome 浏览器（版本 88+）
- Windows、macOS 或 Linux 系统

### 📦 第一步：获取安装包
1. 运行 `npm run build` 生成最新的 `dist/`
2. 或获取打包好的压缩文件 `lite-article-exporter-v1.0.0.zip` 并解压

### 🔧 第二步：启用开发者模式
1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 右上角开启“开发者模式”开关

### ⬇️ 第三步：加载扩展
1. 点击“加载已解压的扩展程序”
2. 选择 `dist/` 目录
3. 点击“选择文件夹”完成加载

### ✅ 第四步：验证安装
- 浏览器右上角会出现扩展图标 📄
- 点击图标，看到扩展界面即安装成功

---

## 📖 使用教程
1. 打开需要保存的网页，点击扩展图标
2. 弹窗会自动填入当前页链接并尝试提取正文
3. 勾选需要的导出格式（Markdown、Word、PDF、TXT）
4. 点击“一键下载”，文件将以 `YYYYMMDD-标题` 命名保存到下载目录
5. 在“下载记录”中可搜索或清空历史记录

---

## ❓ 常见问题

**Q：提示“无法加载扩展程序”？**  
A：确保选择的是解压后的 `dist/` 目录，而不是压缩包。

**Q：提取不到文章内容？**  
A：检查链接是否为 http/https 协议，部分网站可能阻止远程抓取，可在浏览器中打开后再试。

**Q：PDF 或 Word 排版不完整？**  
A：复杂页面可能需要手动调整，建议导出 Word 后再编辑。

**Q：扩展没有响应？**  
A：在 `chrome://extensions/` 中点击“重新加载”，或重新运行 `npm run build` 后再加载。

---

## 🔄 更新扩展
1. 在项目目录执行 `npm run build`
2. 在 `chrome://extensions/` 中点击“重新加载”
3. 如从压缩包升级，可删除旧版本后重新加载新目录

---

## 🛡️ 隐私和权限
- **本地处理**：提取和导出过程完全在本地完成
- **权限说明**：
  - `downloads`：触发文档下载
  - `storage`：保存导出偏好与历史
  - `tabs`：读取当前活动页地址
  - `host_permissions: <all_urls>`：抓取网页内容
